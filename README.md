# ZUT Informatyka Projects

This repository contains programming assignments and study projects completed during computer science studies. It brings together implementations of classic data structures, algorithms, search techniques, optimization methods, and low-level system programming exercises written in C, C++, and Python.

The collection focuses on learning by building core mechanisms from scratch, including custom containers, tree structures, graph algorithms, sorting methods, process and signal handling, multithreading, and selected artificial intelligence techniques.

## Repository Structure

### `C++`
Projects in this directory focus mainly on algorithms and data structures implemented manually, without relying on high-level abstractions. The repository includes, among others:

- Dynamic arrays, linked lists, binary heaps, hash tables, binary search trees, and red-black trees
- Sorting-related exercises, including heap-based sorting and bucket/counting sort implementations
- Graph and search algorithms such as Kruskal's algorithm, Best First Search, and N-Queens solvers using DFS and BFS

### `C`
Projects in this directory are centered around systems programming and lower-level operating system concepts, including:

- Signal handling and parent-child process coordination
- POSIX multithreading
- Custom memory management utilities
- Shared library exercises
- Time measurement and user/system information utilities

### `Python`
Projects in this directory cover selected AI and optimization topics, including:

- A custom Naive Bayes classifier with experiments on discretized datasets
- An evolutionary algorithm for the N-Queens problem

## What This Repository Contains

The projects collected here cover topics such as:

- Data structures implemented from scratch
- Sorting and searching techniques
- Tree and graph algorithms
- State-space search and heuristic methods
- Optimization and evolutionary computation
- Basic machine learning experiments
- Operating system and systems programming exercises
- Concurrent and process-based programming

## Notes

- Most directories are independent student exercises or small standalone projects.
- In many cases, each project is organized around a `main` source file plus supporting headers or helper modules.
- Some tasks include sample input files or Makefiles, depending on the language and course requirements.
